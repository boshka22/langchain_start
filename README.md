# Fraud Detection Chatbot

Антифрод система с RAG (Retrieval-Augmented Generation) для анализа банковских транзакций.
Бот принимает описание транзакции в свободной форме, ищет релевантные правила в базе знаний
и возвращает структурированный результат с оценкой риска.

## Возможности

- Анализ транзакций в режиме реального времени
- RAG — поиск релевантных правил из базы знаний мошенничества
- Структурированный JSON ответ с оценкой риска от 1 до 10
- Три уровня риска: низкий (approve), средний (review), высокий (block)
- Память диалога — бот помнит контекст предыдущих транзакций
- Кэширование векторной БД — быстрый старт при повторных запусках

## Стек

- Python 3.10+
- LangChain
- LangChain Groq (LLM)
- FAISS (векторная база данных)
- HuggingFace Embeddings (sentence-transformers)
- Pydantic

## Установка

```bash
git clone https://github.com/твой-username/fraud-detection-chatbot
cd fraud-detection-chatbot
pip install -r requirements.txt
```

Создай файл `.env` в корне проекта:

```
GROQ_API_KEY=твой-ключ
```

Получить бесплатный ключ можно на [console.groq.com](https://console.groq.com)

## Запуск

```bash
python main.py
```

## Структура ответа

```json
{
    "risk_score": 9,
    "is_suspicious": true,
    "reason": "Транзакция выше 10 000 рублей, ночное время, страна высокого риска",
    "action": "block"
}
```

## Как работает RAG

```
documents.txt → разбивка на чанки → векторизация → FAISS (кэш на диске)
                                                          |
Транзакция пользователя → векторизация → поиск похожих правил
                                                          |
                                    Правила + Транзакция → LLM → JSON ответ
```

## База знаний

Правила обнаружения мошенничества хранятся в `documents.txt`.
Для обновления правил — отредактируй файл и удали папку `fraud_vectorstore`.
При следующем запуске БД пересоздастся автоматически.
